Antes de adentrarnos al mundo del HTML5, primero veamos que significa en si HTML. HTML corresponde a Hipertexto Markup Language , lo que significa:

  • HyperText: corresponde a textos que enlaza con otros contenidos.
  • Markup: Siendo las etiquetas.
  • Lenguage: Lenguage, traduce al español lenguaje, lo que recalca que el HTML es un lenguaje y tiene reglas.


Así que podemos decir que HTML5, es un lenguaje que nos ayuda a estructurar nuestros proyectos o sitios web, a través de etiquetas.

Las etiquetas nos ayudan a crear elementos HTML a través fragmentos de código. Veamos su estructura:





  • Toda etiqueta debe abrirse y cerrarse,a excepción de las etiquetas Self closing que solo necesitan de sí mismas para definirse, las demás, se abren con dos diplé “<>” y se cierran con dos diplé y una barra diagonal.

  • El nombre de las etiquetas, nos ayuda a determinar su función.
  • Los atributos nos ayudan a configurar elementos según lo que deseemos.



Ahora, que ya sabemos que es una etiqueta, veamos una estructura básica HTML5 y su explicación:







  • < DOCTYPE html>: Significa que trabajaremos con HTML.
  • < html lang="en">: Define el lenguaje a utilizar, para cambiar a español ponemos “Es”.
  • < head>: Características del proyecto
  • < meta charset="UTF-8">: Permite el uso de caracteres especiales, tales como : ñ, *, #, @, ETC.
  • < meta name="viewport" content="width=device-width, initial-scale=1.0">: >Nos permite adaptar nuestro código a móviles.
  • < title>: Titulo de nuestro documento HTML, es el que aparecerá en la pestaña del navegador.
  • < body>: Cuerpo y estructura de nuestro proyecto.
