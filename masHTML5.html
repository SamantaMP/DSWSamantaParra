Etiquetas


Etiquetas de contenido: Las etiquetas de contenido son usadas para agrupar contenido dentro de ellas, tienen importancia semántica, y pueden definir secciones, pies de página y menús de navegación, Etc.

  • < nav>: Menù de navegaciòn.
  • < section>: Define seccion del documento HTML.
  • < div>: Agrupar contenido
  • < header>: Define un encabezado en HTML, puede ser de secciones o de la pàgina web.
  • < main>: define contenido principal del documento html. Solo debe haber uno por documento.
  • < footer>: Define pie de página.
  • < aside>: Define sección relacionada con el contenido del documento HTML.
  • < article>: Define una sección independiente del contenido que puede existir por sí mismo (artículos, blog, revista ETC).




Etiquetas de texto: Las etiquetas de texto nos ayudan a mantener un orden en los textos y a que estos tengan una semántica apropiada.



  • < em>: Letra cursiva para dar énfasis.
  • < i>: Letra itálica.
  • < span>: Contenido en línea recta.
  • < mark>: Resalta el texto por su relevancia, como un resaltador.
  • < b>: Texto en negrita.
  • < strong>: Dar importancia a texto en negrita.
  • < u>: Subrayar texto.
  • < small>: Texto menos relevante, texto pequeño.
  • < s>: Tacha el texto.
  • < cities>: Citamos proyecto creativo, en cursiva.
  • < sub> y < sup>: Representa, un subíndice, potencias, ecuaciones, ETC.
  • < br> : Es una etiqueta SelfClosing, no tiene cierre, representa un salto de línea.
  • < lorem> : Etiqueta generadora de texto automático.
  • < abrr>: Abreviar palabras, no la muestra completa, a menos que se ponga el mouse sobre ella


Listas y Tablas:



  • < li>: Agregar, elementos de una lista
  • < ol>: Listas ordenadas en orden numérico.
  • < ul>: Listas en desorden.
  • < table>: Iniciar tabla.
  • < caption>: Nombre o titulo de la tabla.
  • < tr>: Define fila de celdas en un tabla.
  • < th>: Celdas de encabezado, apreceran en negrita.
  • < td>: Celda que contiene datos.



Audio y Video: Las etiquetas de < audio> y < video>, nos ayudaran a poner recursos de audio y video en nuestro proyecto HTML. Estos cuentan con atributos para controlar su funciòn, son:



  • < Controls>: Controles.
  • < muted>: Callar video.
  • < loop>: Reproducciòn en loop, en infinito.
  • < autoplay>: Auto reproducir.
  • < poster>: Imagen al video.
